# Example Concept

> Concepts are cross-cutting ideas shared by several projects — an auth model, a
> booking flow, a data pipeline. Project pages link to them; that's the graph.

Describe the concept once here, link it from every project page that touches it, and the
agent gets the shared context without you repeating it.
