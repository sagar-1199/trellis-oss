# Backlog

Your tasks. The loop picks the top unchecked one. Each task is a Markdown checkbox
with inline fields the loop parses. Check the box (or the loop does) when done.

## Task format
```
- [ ] (priority:: 1) (repo:: example-app) Short imperative title
    - verify:: npm run build && npm run test   # blank = the project page's default
    - accept:: what "done" means, concretely and checkably
    - notes:: optional context, links, [[Concept Page]]
```
